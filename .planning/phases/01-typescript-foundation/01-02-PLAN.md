For EACH error location, add a null guard BEFORE the property access:
```typescript
// Pattern for array index access:
const winner = candidates[0];

if (!effectiveEntry) continue;  // or return
// now effectiveEntry is narrowed
```

Read the full file to identify all 19 error locations. Apply the guard pattern at each one. Use `continue` inside loops, `return` inside functions, or `return []`/`return undefined` when the function has a return type.
