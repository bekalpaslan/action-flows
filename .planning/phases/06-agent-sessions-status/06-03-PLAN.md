Create `packages/backend/src/services/sessionHealthMonitor.ts`:

The SessionHealthMonitor class implements heartbeat-based health detection per SESSION-04 (< 30s latency) and resurrection per SESSION-05 (from local conversation logs).

```typescript
import { getSessionInfo, getSessionMessages, listSessions } from '@anthropic-ai/claude-agent-sdk';
import { backendSessionStore } from './sessionStore.js';
import type { SessionManager } from './sessionManager.js';
```

**Class: `SessionHealthMonitor`**

Constructor: `(manager: SessionManager, projectDir: string)`
- Store manager and projectDir references
- Set `HEARTBEAT_INTERVAL_MS = 15_000` (15s interval = worst case 30s detection per SESSION-04)
- Set `STALE_THRESHOLD_MS = 30_000` (session considered stale if no activity for 30s)
- Set `heartbeatTimer: ReturnType<typeof setInterval> | null = null`
- Log prefix: `[SessionHealth]`

**`start(): void`**
- Clear any existing timer
- Start a `setInterval` at `HEARTBEAT_INTERVAL_MS` that calls `checkAllSessions()`
- Log `[SessionHealth] Health monitor started (${HEARTBEAT_INTERVAL_MS}ms interval)`

**`stop(): void`**
- Clear the interval timer
- Log `[SessionHealth] Health monitor stopped`

**`private async checkAllSessions(): Promise<void>`**
- Iterate all sessions from `backendSessionStore.getAll()`
- For each session with status `idle` or `running`:
  - Check if `lastActivity` is older than `STALE_THRESHOLD_MS`
  - If stale:
    - Try to verify session still exists via `getSessionInfo(session.sessionId, { dir: projectDir })`
    - If session info returns null or throws: session is dead
    - Call `handleDisconnectedSession(session.workbenchId, session.sessionId)`
  - If not stale: session is healthy, no action
- Wrap each check in try/catch (one session's failure shouldn't block others)

**`private async handleDisconnectedSession(workbenchId: string, sessionId: string | null): Promise<void>`**
- Log `[SessionHealth] Detected disconnection for ${workbenchId}`
- Set session status to `error` with error message `"Session disconnected — attempting resurrection"`
- Broadcast status update via the manager's broadcastStatus (call `manager.stopSession(workbenchId)` to clean up, then attempt resurrection)
- Call `attemptResurrection(workbenchId, sessionId)`

**`private async attemptResurrection(workbenchId: string, sessionId: string | null): Promise<void>`** (per SESSION-05)
- Log `[SessionHealth] Attempting resurrection for ${workbenchId}`
- If `sessionId` is not null:
  - Try to resume the session: call `manager.startSession(workbenchId)` (which will use the stored sessionId for resume)
  - If resume succeeds: log success, return
  - If resume fails: fall through to discovery
- Discover recent sessions: call `listSessions({ dir: projectDir, limit: 5 })`
- Look for a session matching this workbench (by metadata or most recent)
- If found: update sessionId in backendSessionStore, call `manager.startSession(workbenchId)`
- If not found: log `[SessionHealth] No recoverable session for ${workbenchId}`, leave status as `error`
- All operations wrapped in try/catch with appropriate logging

**`getHealthStatus(): Record<string, { status: string; lastCheck: string | null; healthy: boolean }>`**
- Return a summary of all session health statuses for API consumers

Export:
```typescript
export let sessionHealthMonitor: SessionHealthMonitor | null = null;
export function initSessionHealthMonitor(manager: SessionManager, projectDir: string): SessionHealthMonitor {
  sessionHealthMonitor = new SessionHealthMonitor(manager, projectDir);
  return sessionHealthMonitor;
}
```

2. **Extend `packages/backend/src/ws/handler.ts`** to handle session lifecycle commands.

   Add imports at the top:
   ```typescript
   import { sessionManager } from '../services/sessionManager.js';
   ```

   In the `switch (message.type)` block, add new cases before the `default` case:

   ```typescript
   case 'session:start': {
     const { workbenchId } = (message as any).payload ?? {};
     if (!workbenchId || !sessionManager) {
       ws.send(JSON.stringify({ type: 'error', payload: 'Invalid session:start — workbenchId required' }));
       break;
     }
     try {
       await sessionManager.startSession(workbenchId);
       console.log(`[WS] Session started for workbench ${workbenchId} by client ${clientId}`);
     } catch (err) {
       console.error(`[WS] Session start failed for ${workbenchId}:`, err);
       ws.send(JSON.stringify({ type: 'error', payload: `Session start failed: ${(err as Error).message}` }));
     }
     break;
   }

   case 'session:stop': {
     const { workbenchId } = (message as any).payload ?? {};
     if (!workbenchId || !sessionManager) {
       ws.send(JSON.stringify({ type: 'error', payload: 'Invalid session:stop — workbenchId required' }));
       break;
     }
     try {
       await sessionManager.stopSession(workbenchId);
       console.log(`[WS] Session stopped for workbench ${workbenchId} by client ${clientId}`);
     } catch (err) {
       console.error(`[WS] Session stop failed for ${workbenchId}:`, err);
       ws.send(JSON.stringify({ type: 'error', payload: `Session stop failed: ${(err as Error).message}` }));
     }
     break;
   }

   case 'session:switch': {
     const { newWorkbenchId, previousWorkbenchId } = (message as any).payload ?? {};
     if (!newWorkbenchId || !previousWorkbenchId || !sessionManager) {
       ws.send(JSON.stringify({ type: 'error', payload: 'Invalid session:switch — newWorkbenchId and previousWorkbenchId required' }));
       break;
     }
     sessionManager.handleWorkbenchSwitch(newWorkbenchId, previousWorkbenchId);
     console.log(`[WS] Workbench switch: ${previousWorkbenchId} → ${newWorkbenchId} by client ${clientId}`);
     break;
   }

   case 'session:history': {
     const { workbenchId } = (message as any).payload ?? {};
     if (!workbenchId || !sessionManager) {
       ws.send(JSON.stringify({ type: 'error', payload: 'Invalid session:history — workbenchId required' }));
       break;
     }
     try {
       const history = await sessionManager.getSessionHistory(workbenchId);
       ws.send(JSON.stringify({
         channel: '_system',
         type: 'session:history',
         payload: { workbenchId, messages: history },
         ts: new Date().toISOString(),
       }));
     } catch (err) {
       console.error(`[WS] Session history failed for ${workbenchId}:`, err);
     }
     break;
   }
   ```
