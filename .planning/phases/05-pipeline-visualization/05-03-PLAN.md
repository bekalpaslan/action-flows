**useElapsedTime.ts** -- Live elapsed time counter for running nodes:

Import `useState, useEffect, useRef` from `react`.

Export function `useElapsedTime(startedAt: string | null, isRunning: boolean): number | null`:
1. `const [elapsed, setElapsed] = useState<number | null>(null)`
2. `const intervalRef = useRef<ReturnType<typeof setInterval> | null>(null)`
3. useEffect:
   - If NOT running or no startedAt: clear interval, return cleanup.
   - Parse start time: `const start = new Date(startedAt).getTime()`
   - Immediate first tick: `setElapsed(Date.now() - start)`
   - Set interval at 1000ms: `intervalRef.current = setInterval(() => setElapsed(Date.now() - start), 1000)`
   - Cleanup: clear interval.
4. Dependencies: `[isRunning, startedAt]`
5. Return elapsed.

**usePipelineEvents.ts** -- WebSocket subscription that maps chain/step events to store updates:

Import `useEffect, useRef, useCallback` from `react`.
Import `wsClient` from `@/lib/ws-client`.
Import `usePipelineStore` from `@/stores/pipelineStore`.
Import `layoutPipeline` from `@/lib/pipeline-layout`.
Import `type WSEnvelope` from `@afw/shared`.
Import `type WorkbenchId` from `@/lib/types`.

Export function `usePipelineEvents(workbenchId: WorkbenchId): void`:

1. Get store actions via narrow selectors (one per action to avoid selector-granularity pitfall):
   ```typescript
   const initChain = usePipelineStore((s) => s.initChain);
   const updateNodeStatus = usePipelineStore((s) => s.updateNodeStatus);
   const updateEdgeStatus = usePipelineStore((s) => s.updateEdgeStatus);
   const clearPipeline = usePipelineStore((s) => s.clearPipeline);
   ```

2. Use a ref to buffer incoming events for ~100ms batching (per RESEARCH.md Pitfall 3):
   ```typescript
   const bufferRef = useRef<WSEnvelope[]>([]);
   const rafRef = useRef<number | null>(null);
   ```

3. Create `processBuffer` function using useCallback that iterates over buffered envelopes and dispatches to store:
   For each envelope in buffer, switch on `envelope.type`:

   - `'chain:compiled'`: Extract `payload` as `ChainCompiledEvent`. If `payload.steps` exists and is non-empty, call `layoutPipeline(payload.steps)` to get nodes/edges. Then call `initChain(workbenchId, payload.chainId ?? 'unknown', nodes, edges)`.

   - `'step:spawned'`: Extract `payload` as `StepSpawnedEvent`. Call `updateNodeStatus(workbenchId, String(payload.stepNumber), { name: payload.action ?? 'Unknown', agentModel: payload.model ?? null, description: payload.description ?? null, inputs: payload.inputs ?? null })`.

   - `'step:started'`: Extract `payload` as `StepStartedEvent`. Call `updateNodeStatus(workbenchId, String(payload.stepNumber), { status: 'running', startedAt: payload.startedAt })`. Also update incoming edge to active: find edge with target = `String(payload.stepNumber)` and call `updateEdgeStatus(workbenchId, edgeId, { data: { status: 'active' } })`.

   - `'step:completed'`: Extract `payload` as `StepCompletedEvent`. Call `updateNodeStatus(workbenchId, String(payload.stepNumber), { status: 'complete', elapsedMs: payload.duration, result: payload.result ?? null, fileChanges: payload.fileChanges ?? null })`. Update incoming edge: `{ data: { status: 'completed' } }`.

   - `'step:failed'`: Extract `payload` as `StepFailedEvent`. Call `updateNodeStatus(workbenchId, String(payload.stepNumber), { status: 'failed', error: payload.error ?? null, suggestion: payload.suggestion ?? null })`. Update incoming edge: `{ data: { status: 'failed' } }`.

   - `'step:skipped'`: Extract `payload` as `StepSkippedEvent`. Call `updateNodeStatus(workbenchId, String(payload.stepNumber), { status: 'skipped' })`.

   - `'chain:completed'`: After a 5-second delay (setTimeout), call `clearPipeline(workbenchId)`. The delay is per UI-SPEC chain completion transition. If a new `chain:compiled` arrives during the 5s window, the `clearPipeline` timeout should be cancelled. Use a ref to track the timeout ID.

   Clear buffer after processing.

4. useEffect subscribes to the workbench channel:
   ```typescript
   const unsub = wsClient.subscribe(workbenchId, (envelope: WSEnvelope) => {
     bufferRef.current.push(envelope);
     if (rafRef.current === null) {
       rafRef.current = requestAnimationFrame(() => {
         processBuffer();
         rafRef.current = null;
       });
     }
   });
   return () => {
     unsub();
     if (rafRef.current !== null) cancelAnimationFrame(rafRef.current);
   };
   ```

5. Dependencies: `[workbenchId, initChain, updateNodeStatus, updateEdgeStatus, clearPipeline, processBuffer]`

**Note on edge status updates:** The hook needs to find the edge ID for a given target node. It should read the current pipeline state from the store: `usePipelineStore.getState().pipelines.get(workbenchId)?.edges.find(e => e.target === String(stepNumber))?.id`. Use `.getState()` (direct store access) rather than a selector to avoid re-render dependencies.
