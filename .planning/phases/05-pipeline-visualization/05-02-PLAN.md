**StepNode.tsx** -- Custom @xyflow/react node for chain steps (rounded rectangle, per D-01 rich nodes):

Import `Node, NodeProps, Handle, Position` from `@xyflow/react`, `cn` from `@/lib/utils`, `cva` from `class-variance-authority`, lucide icons `CheckCircle2, XCircle, Loader2, SkipForward, Circle` from `lucide-react`, and `StepNodeData` from `@/lib/pipeline-types`.

Import `'../pipeline/pipeline.css'` to ensure CSS keyframes are loaded.

Define `type StepNodeType = Node<StepNodeData, 'step'>` (or import from pipeline-types).

Create CVA variant for the left border color based on status:
```typescript
const statusBorder = cva('border-l-[3px]', {
  variants: {
    status: {
      pending: 'border-l-border',
      running: 'border-l-accent',
      complete: 'border-l-success',
      failed: 'border-l-destructive',
      skipped: 'border-l-border',
    },
  },
});
```

Create STATUS_ICONS map:
```typescript
const STATUS_ICONS = {
  pending: Circle,
  running: Loader2,
  complete: CheckCircle2,
  failed: XCircle,
  skipped: SkipForward,
} as const;
```

Export function `StepNode({ data, selected }: NodeProps<StepNodeType>)`:
- Outer div: `cn('min-w-[160px] rounded-md bg-surface-2 border border-border p-3 shadow-sm hover:shadow-md hover:border-border-strong transition-shadow cursor-pointer', statusBorder({ status: data.status }), data.status === 'running' && 'animate-pipeline-pulse', selected && 'ring-2 ring-accent shadow-glow-focus')`
- Handle target: `<Handle type="target" position={Position.Left} className="!bg-border-strong !w-2 !h-2" />`
- Row 1: flex items-center gap-2 with Icon (16px, status-dependent color classes: running = `animate-spin text-accent`, complete = `text-success`, failed = `text-destructive`, pending/skipped = `text-text-muted`) and step name (text-caption font-semibold truncate max-w-[120px])
- Row 2: elapsed time (text-caption text-text-dim mt-1). Format: if `data.elapsedMs != null`, use `formatElapsed(data.elapsedMs)`, else `'--'`
- Row 3: agent model if present (text-caption text-text-muted truncate)
- Handle source: `<Handle type="source" position={Position.Right} className="!bg-border-strong !w-2 !h-2" />`

Helper `formatElapsed(ms: number): string`:
- If ms < 60000: `${Math.round(ms / 1000)}s`
- Else: `${Math.floor(ms / 60000)}m ${Math.round((ms % 60000) / 1000)}s`

Export `formatElapsed` for reuse in the drawer.

**PipelineEdge.tsx** -- Custom edge with animated flow for active paths:

Import `BaseEdge, getSmoothStepPath, type EdgeProps` from `@xyflow/react`. Import pipeline.css.

Export function `PipelineEdge(props: EdgeProps)`:
- Destructure `sourceX, sourceY, targetX, targetY, sourcePosition, targetPosition, data, style, markerEnd` from props.
- Call `const [edgePath] = getSmoothStepPath({ sourceX, sourceY, targetX, targetY, sourcePosition, targetPosition })`.
- Determine edge status from `data?.status` (type as `'default' | 'active' | 'completed' | 'failed'`).
- Style the edge:
  - default: `stroke: var(--color-border-strong)`, strokeWidth 2
  - active: `stroke: var(--color-accent)`, strokeWidth 2, add className `animate-pipeline-edge-flow`
  - completed: `stroke: var(--color-success)`, strokeWidth 2, opacity 0.5
  - failed: `stroke: var(--color-destructive)`, strokeWidth 2, opacity 0.5
- Return `<BaseEdge path={edgePath} style={computedStyle} className={edgeClassName} markerEnd={markerEnd} />`

**Define module-level constants:**

```typescript
export const nodeTypes = { step: StepNode, gate: GateNode } as const;
export const edgeTypes = { pipeline: PipelineEdge } as const;
```
