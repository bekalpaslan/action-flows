    1. Get active workbench: `const workbenchId = useUIStore(s => s.activeWorkbench);`
    2. Get workbench meta: `const workbenchMeta = WORKBENCHES.find(w => w.id === workbenchId)!;`

    4. Get chat state: `const chat = useChatStore(s => s.getChat(workbenchId));`
    5. Call useChatMessages() -- once for WebSocket subscription.
