# project/

Project-specific flows for ActionFlows Dashboard implementation work.

- audit-and-fix/ → QA with remediation
- bug-triage/ → Systematic bug investigation and fixes
- cli-integration-test/ → CLI integration testing
- code-and-review/ → Standard code implementation + review
- contract-compliance-audit/ → Contract format compliance checking
- contract-format-implementation/ → Implement new contract formats
- contract-index/ → Contract catalog management
- design-system-sync/ → Sync design tokens from Figma variables
- design-to-code/ → Convert Figma designs to React components
- e2e-chrome-mcp/ → Browser E2E testing with Chrome MCP
- intel-analysis/ → Intelligence gathering and analysis
- post-completion/ → Post-task documentation updates
- test-coverage/ → Test coverage analysis and improvement
